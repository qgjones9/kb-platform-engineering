# [nginx](https://nginx.org/en/docs/)

NGINX is a high-performance web server, reverse proxy, and load balancer. It is widely used to serve static content, act as an HTTP/HTTPS reverse proxy, and distribute traffic across backend servers.

## Key capabilities

- **Web server** — Serves static and dynamic content; supports HTTP, HTTPS, and (with recent versions) [QUIC and HTTP/3](https://nginx.org/en/docs/).
- **Reverse proxy** — Proxies requests to backend servers; supports [WebSocket proxying](https://nginx.org/en/docs/) and [HTTPS server configuration](https://nginx.org/en/docs/).
- **Load balancer** — [HTTP load balancing](https://nginx.org/en/docs/) with configurable upstream groups and health checks.
- **Kubernetes** — [NGINX Ingress Controller](https://nginx.org/en/docs/) and Gateway API implementation for ingress and routing in Kubernetes.
